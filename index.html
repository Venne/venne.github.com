  • Prohlížet na GitHubu Prohlížet dokumentaci

    Jednoduchý redakční systém pro Vaše weby kombinující výhody Nette frameworku a uživatelsky přívětivého systému.

    Instalace z archivu

    Obsah archivu rozbalte do veřejného adresáře webového serveru.

    Stáhnout zip archiv Stáhnout tgz archiv

    Instalace ze zdrojových kódů

    composer create-project venne/sandbox:2.0.x-dev myApp
cd myApp
composer require venne/cms-module:2.0.x [--prefer-dist]
php www/index.php venne:module:update
php www/index.php venne:module:install cms [--noconfirm]

    Instalace modulů

    composer require venne/<name>-module:2.0.x [--prefer-dist]

    Aktivace modulů

    # prohledá nové moduly na filesystému
php www/index.php venne:module:update

# výpis dostupných modulů
php www/index.php venne:module:list

# nainstaluje modul
php www/index.php venne:module:install <name> [--noconfirm]
